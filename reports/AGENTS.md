# Reports Directory

This directory contains benchmark reports and their associated assets.

## Style Guide

1. Avoid em dashes (—). Prefer commas and periods.
2. Use semicolons sparingly. Prefer separate sentences.
4. Use colons sparingly. Prefer separate sentences. Typically, colons should only be used to introduce a list or a definition, not within a sentence.
3. Use Title Case for all headings. Make sure to keep prepositions and articles lowercase, as appropriate in headings.
